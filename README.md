### How to run:
  - Install [MinGW](https://sourceforge.net/projects/mingw/)
  
  - ```gcc -g shell.c -o shell.exe```
  
  - ```./shell.exe```

### Available commands:
  - ```ls```
  - ```cd```
  - ```mkdir```
  - ```rm```
  - ```clear```
  - ```touch```
  - ```echo```
  - ```cat```
  - ```history```
  - ```cp```
  - ```mv```
